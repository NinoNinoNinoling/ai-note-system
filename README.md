# 🧠 AI 지능형 노트 시스템

> LangChain + Claude + RAG 기반 차세대 노트 관리 플랫폼

## 🎯 프로젝트 개요

**개발 기간:** 4일  
**핵심 기술:** LangChain + Claude 3.5 Sonnet + RAG + Vue.js  
**목적:** AI 기반 지식 관리 시스템으로 노트 작성, 검색, 분석을 통합한 혁신적 학습 도구

### 4. 실시간 데이터 시각화
Chart.js를 활용한 노트 생성 추세, 태그 분포, 시스템 성능 지표의 실시간 모니터링

Flask의 애플리케이션 팩토리 패턴과 Vue.js의 Composition API를 활용한 확장 가능한 설계

## 💻 실행 방법

### 1. 환경 설정
```bash
# 백엔드 설정
cd backend
pip install -r requirements.txt
cp .env.example .env

# .env 파일에 Claude API 키 설정
ANTHROPIC_API_KEY=your-claude-api-key

# 프론트엔드 설정
cd frontend/ai-note-frontend
npm install
```

### 2. 실행
```bash
# 프론트엔드 빌드
cd frontend/ai-note-frontend && npm run build

# 백엔드 실행 (Vue 정적 파일 포함 서빙)
cd backend && flask run
```

### 3. 접속
- **메인 페이지**: http://localhost:5000
- **API 문서**: http://localhost:5000/api/info  
- **시스템 상태**: http://localhost:5000/health
- **대시보드**: 실시간 통계 및 차트 확인
- **노트 작성**: 마크다운 편집기 + AI 연동
- **AI 채팅**: RAG 기반 지능형 대화
- **고급 검색**: 텍스트 + 의미 검색
